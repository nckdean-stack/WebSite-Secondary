# WebSite-Secondary

This repository contains a minimal starter website created by an automated assistant.

Files added:
- index.html — main HTML page
- styles.css — basic styling
- script.js — minimal JS

To view locally: open `index.html` in a browser.

To publish via GitHub Pages: go to repository Settings → Pages and enable Pages from the `main` branch (root). After a minute the site will be available at `https://<your-username>.github.io/WebSite-Secondary/`.

Customize the content in `index.html` to make the site your own.
